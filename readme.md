# Urban Traffic Intelligence Dashboard

A Tableau-based analytics project that analyzes city traffic patterns, accident hotspots, and commute efficiency.

## Objective
Identify congestion zones, peak travel hours, and weather impact to support smart urban planning.

## Dataset
- Traffic events by intersection
- Time-of-day patterns
- Weather conditions
- Accident frequency

## Key Insights
- 70% congestion concentrated in top 5 intersections
- Rain increases commute time by 22%
- Morning peak worse than evening peak

## Tools Used
- Tableau
- CSV datasets
- Data modeling & visualization
